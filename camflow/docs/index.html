CamFlow

Hold a key. Talk. Your words appear — in any app on your Mac.
100% on-device with Whisper. No account, no cloud, no subscription.

Install — paste into Terminal

git clone https://github.com/Camcassar/CamFlow.git && cd CamFlow && ./run.sh

First-time setup

  1. 1The first run installs everything and downloads the Whisper model (~1.5 GB, one time — fully offline after that).
  2. 2Grant Microphone, Input Monitoring, and Accessibility to your terminal when macOS asks (System Settings → Privacy & Security).
  3. 3Wait for the 🎤 in your menu bar, click into any text field, then hold Right ⌥ Option and speak. Release — the text appears.
